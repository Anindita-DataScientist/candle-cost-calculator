# 🕯️ Candle Cost Calculator

A premium candle pricing calculator built using HTML, CSS, and JavaScript to help handmade candle businesses calculate:

* Cost per candle
* Recommended selling price
* Profit per candle
* Batch profit
* Profit margin %

This tool is designed for small candle businesses, handmade product sellers, and candle studio owners who want a clean and professional pricing workflow.

---

# ✨ Features

✅ Real-time cost calculation
✅ Recommended selling price using the **2.5× pricing rule**
✅ Profit per candle calculation
✅ Batch profit calculation
✅ Profit margin percentage
✅ Dynamic cost distribution bars
✅ Responsive premium UI
✅ Lightweight and fast
✅ No backend required

---

# 🧮 Calculation Logic

## 1. Total Cost Per Candle

The calculator sums all material and operational costs:

* Wax
* Glass/Jar
* Wick
* Fragrance Oil
* Color/Dye
* Electricity
* Labour
* Packaging

---

## 2. Recommended Selling Price

Uses the standard handmade business pricing formula:

```text
Recommended Price = Total Cost × 2.5
```

---

## 3. Profit Per Candle

```text
Profit Per Candle = Selling Price − Total Cost
```

---

## 4. Batch Profit

```text
Batch Profit = Profit Per Candle × Quantity
```

---

## 5. Profit Margin %

```text
Profit Margin % = (Profit ÷ Selling Price) × 100
```
